## Profile

I work on performance-oriented software at the intersection of scientific computing, GPU programming, and applied machine learning. My projects focus on clear implementations, reproducible experiments, and practical validation of algorithms.

Current areas of interest:

- CUDA kernel optimization and GPU operator performance
- Numerical methods for PDEs and scientific computing workflows
- Parallel simulation, including lattice Boltzmann methods
- Neural networks, AI agents, and algorithmic project work

## Technical Focus

| Domain | Work |
| --- | --- |
| GPU Computing | CUDA kernels, memory access patterns, operator benchmarking |
| Numerical Computing | Poisson and heat equation solvers, BLAS/LAPACK workflows, validation tools |
| Scientific Simulation | D2Q9 lattice Boltzmann method, Karman vortex street, parallel experiments |
| Applied AI | Neural networks from scratch, MNIST classification, game AI projects |
