## 个人简介

我关注面向性能的软件开发，方向集中在科学计算、GPU 编程和应用机器学习的交叉领域。我的项目强调清晰实现、可复现实验和算法的实际验证。

当前关注方向：

- CUDA kernel 优化和 GPU 算子性能
- 面向偏微分方程的数值方法与科学计算流程
- 并行模拟，尤其是格子玻尔兹曼方法
- 神经网络、AI 智能体和算法项目

## 技术方向

| 方向 | 内容 |
| --- | --- |
| GPU 计算 | CUDA kernel、内存访问模式、算子 benchmark |
| 数值计算 | Poisson/热方程求解器、BLAS/LAPACK 工作流、验证工具 |
| 科学模拟 | D2Q9 格子玻尔兹曼方法、卡门涡街、并行实验 |
| 应用 AI | 从零实现神经网络、MNIST 分类、游戏 AI 项目 |
